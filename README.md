# Image-Pre-Processing-Excercise
Basic Exercise for practice of Computer Vision
1. Draw a rectangle, circle and line on a white image using OpenCV.
White image has to be created using numpy. Any height &
width.
Output file name: result1.png
2. Folder “input1” contains images of physical boxes with
transparent backgrounds. Convert the transparent
background to black color without affecting the physical
boxes.
Output stored in files : result2.png & result3.png
